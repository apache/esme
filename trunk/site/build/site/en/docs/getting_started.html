ESME Desktop Flex/AIR Client

ESME Desktop is a desktop client for ESME API built with Adobe Flex and Adobe AIR technologies.

Installation

Here's how you can install it

  1. Download and install the latest Adobe AIR runtime from http://get.adobe.com/air/
  2. Download the latest released version of ESMEDesktop.air from http://code.google.com/p/esmeproject/downloads/list
  3. Follow the installation wizard to install
