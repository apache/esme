Introduction

Based on its REST-API, there are a variety of clients available for ESME. These clients are not part of the Apache svn but remain in the Google Code repository. A overview of the possible integration patterns is described here.

These clients are usual contributed by the community.

List of actual clients

  1. ABAP
  2. Javascript
  3. PL/SQL
  4. Java for which there is also a client implementation for a SAP NetWeaver implementation
  5. VBA
